# Position: Senior ML Systems Engineer
# Company: Aurora Applied AI

Aurora Applied AI is hiring a Senior ML Systems Engineer to build high-performance inference systems for large language models.

Responsibilities:
- Build and optimize GPU inference pipelines (latency, throughput, memory).
- Productionize model serving across Kubernetes clusters.
- Design scalable APIs for model access with SLOs and observability.
- Lead performance profiling, optimization, and cost reduction.
- Collaborate with applied research to translate prototypes into production systems.

Requirements:
- 5+ years of ML systems engineering or applied research experience.
- Proven experience with GPU kernel optimization, CUDA/Triton, and inference acceleration.
- Experience with distributed systems, Kubernetes, and cloud deployments.
- Strong software engineering skills in Python/C++.
- Track record of shipping ML infrastructure used in production.

Preferred:
- Experience with LLM inference, quantization, and model compression.
- Open-source contributions in ML systems.
